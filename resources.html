Metadata:

Permanent Pixels NFT metadata includes the following variables: description, name, random seed, skew, rarity and image. Below is a breakdown of the different variables.

Description

The description contains a short summary of the NFTs: "Perfectly random, black and white images, with a resolution of 2500 pixels."

Name

"Permanent Pixel" followed by the NFT token number.

Random Seed

Number used to determine image expression. Recorded by the Permanent Pixels team to screen for extremely rare duplicates.

Skew

A black skew indicates that a NFT has more black pixels than white in the image and vice versa. If the skew has a "none" value, there is an equal number of black and white pixels in the image.

Rarity

A number between 0 and 2500. The higher the number, the more rare a NFT is. The difficulty of getting a high rarity number increases on an exponential scale.

Image

The on-chain SVG image data.
